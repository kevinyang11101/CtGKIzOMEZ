# 企业OA管理系统 java566

## 项目概述

企业OA管理系统是基于Springboot+Vue的前后端分离项目。该项目采用MVC三层架构，结合Spring、SpringBoot、Mybatis及Vue技术栈，搭配Maven构建工具，打造了一个高效、易用的企业办公自动化解决方案。

## 技术栈

- **后端：**
  - Spring
  - SpringBoot
  - Mybatis
- **前端：**
  - Vue
- **数据库：**
  - MySQL
- **构建工具：**
  - Maven

## 功能模块

系统主要包含以下功能模块：

1. 用户管理
2. 部门管理
3. 工作流程管理
4. 通知公告管理
5. 文件管理
6. 日程安排

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- **后端：**
  - JDK 1.8 或以上
  - MySQL 5.7 或以上
- **前端：**
  - 现代浏览器（建议使用Chrome或Firefox）

## 部署步骤

1. 准备好运行环境（JDK、MySQL）
2. 初始化数据库
3. 编译后端代码，部署到服务器
4. 编译前端代码，部署到Web服务器

## 目录结构

```
├── backend               // 后端代码
├── frontend              // 前端代码
└── sql                   // 数据库脚本
```

## 核心亮点

- **前后端分离**：便于管理，易于扩展，提升了开发效率。
- **高效稳定**：采用成熟技术栈，确保系统的高效和稳定性。
